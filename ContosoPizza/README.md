**Contoso Pizza** is a simple **ASP.NET Core Web API project** built to practice creating RESTful endpoints using C# and .NET.  
It provides basic pizza-related operations through HTTP requests and demonstrates how to structure a web service using controllers, models, and services.

This project was developed as part of my learning journey to understand how web APIs are built and consumed in real-world applications.
